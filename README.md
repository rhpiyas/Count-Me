# Count Website

A simple counter website built using **HTML, CSS, and JavaScript**.  
This project allows users to increase the count value by clicking a button.

## 🚀 Features

- Simple and clean user interface
- Click button to increase count
- Dynamic count update using JavaScript
- Responsive basic design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
